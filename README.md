# 🤖 Redouane AI - بوت واتساب ذكي

بوت واتساب متقدم يعتمد على الذكاء الاصطناعي لتقديم خدمات متعددة مع دعم اللغة الدارجة المغربية.

## ✨ المميزات

- 🎨 **توليد الصور**: إنشاء صور بناءً على وصف نصي باستخدام Stable Diffusion
- 👁️ **تحليل الصور**: تحليل الصور وشرحها بالدارجة المغربية
- 💻 **توليد الأكواد**: كتابة أكواد برمجية بلغات مختلفة
- 😂 **نكات مغربية**: الاستمتاع بنكات طريفة بالدارجة
- 💭 **الذاكرة**: حفظ السياق للمحادثات الطويلة
- 🌍 **دعم الدارجة**: واجهة كاملة باللغة الدارجة المغربية

## 🔧 المتطلبات

- Node.js (v14 أو أحدث)
- npm أو yarn
- مفاتيح API:
  - **GROQ_API_KEY**: من [console.groq.com](https://console.groq.com)
  - **HF_TOKEN**: من [huggingface.co](https://huggingface.co)
  - **GEMINI_API_KEY**: (اختياري) من [makersuite.google.com](https://makersuite.google.com)

## 🚀 التثبيت والتشغيل

### 1. استنساخ المشروع
```bash
git clone https://github.com/redouan991/Redouane-Bot-.git
cd Redouane-Bot-
```

### 2. تثبيت التبعيات
```bash
npm install
```

### 3. إعداد متغيرات البيئة
أنشئ ملف `.env` وأضف مفاتيح API الخاصة بك:
```env
GROQ_API_KEY=your_groq_api_key_here
HF_TOKEN=your_huggingface_token_here
GEMINI_API_KEY=your_gemini_api_key_here
```

### 4. تشغيل البوت
```bash
npm start
```

بعدها ستظهر رمز QR في الطرفية - امسحه باستخدام واتساب لربط البوت بحسابك.

## 📝 الأوامر المتاحة

| الأمر | الاستخدام | الوصف |
|------|----------|-------|
| `/start` أو `سلام` | بلا حاجة | عرض رسالة الترحيب والأوامر |
| `/صورة` | `/صورة وصف الصورة` | توليد صورة من الوصف |
| `/حلل` | `صورة + /حلل` | تحليل الصورة المرسلة |
| `/كود` | `/كود python` | كتابة كود برمجي |
| `/نكتة` | بلا حاجة | قول نكتة مضحكة |
| `/صفر` | بلا حاجة | حذف الذاكرة وإعادة تعيين |
| `/help` | بلا حاجة | عرض قائمة الأوامر |
| `/مالك` أو `/بوت` | بلا حاجة | معلومات عن البوت |
